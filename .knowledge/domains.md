# VCT Platform Domains (Bộ nhớ Tên miền)

Đây là quy chuẩn tên miền chính thức của Hệ sinh thái VCT Platform để các Agent sử dụng thống nhất khi viết bài, tạo link, hoặc code API:

- **Website (Trang chủ/Landing Page):** https://vctplatform.vn
- **WebApp (Hệ sinh thái ứng dụng quản trị/ERP/SaaS):** https://app.vctplatform.vn

*Lưu ý (V12 Enforcer): Bất kỳ Agent nào tạo URL điều hướng người dùng đăng nhập hoặc sử dụng phần mềm, bắt buộc dùng `app.vctplatform.vn`. Tuyệt đối không dùng domain cũ của Vercel.*
