# VCT Platform Auto-Knowledge Base

This directory is an automated RAG (Retrieval-Augmented Generation) knowledge base for the AIs working on this platform.

**Rules for Agents:**
1. **Search Before Act:** ALWAYS use local terminal tools to `grep` or search on this directory before starting complex tasks to see if a previous agent left a K.I (Knowledge Item).
2. **Deposit After Act:** When you solve a hard bug, create an architectural pattern, or finalize a complex workflow, write a concise Markdown file summarizing the lesson or API structure here (e.g., `.knowledge/auth-token-gotchas.md`).
